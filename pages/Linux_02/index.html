02.nano 编辑器

# nano 编辑器

本文介绍一下在ubuntu 20.04系统下nano文本编辑器的用法

# 以动手学形式掌握最基本用法

  1. 创建并一个文件
     nano notes.txt
    1
  2. 进入编辑界面，可以在里面随便写一行文字，比如：
    这是关于nano的学习介绍
    1
  3. 保存文件
    • 按Ctrl + O
    • 回车确认文件名 notes.txt
    • 会提示[ Wrote 2 lines] 之类的信息
  4. 退出nano
    • 按Ctrl + X

# 常用快捷键

  • Ctrl + G：打开帮助菜单
  • Ctrl + O：保存（Write Out）
  • Ctrl + X：退出
  • Ctrl + K：剪切一整行
  • Ctrl + U：粘贴
  • Ctrl + W：搜索
  • Ctrl + C：显示光标所在位置（行号/列号）
  • Alt + A：开始选中（标记）
  • Ctrl + _：跳转到指定行
