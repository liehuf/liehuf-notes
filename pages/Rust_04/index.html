04.Rust的控制流

# Rust 的控制流

控制流指的是程序根据条件和循环来决定执行路径。Rust 提供了清晰的条件分支和循环语句，让代码逻辑更容易表达。

# 一、if 表达式

fn main() {
    let n = 7;

    if n < 5 {
        println!("小于5");
    } else if n == 5 {
        println!("等于5");
    } else {
        println!("大于5");
    }
}

let x = if n > 0 { 1 } else { -1 };

# 二、loop 循环

fn main() {
    let mut count = 0;

    loop {
        count += 1;
        if count == 3 {
            println!("跳出循环");
            break;
        }
    }
}

  • loop 会无限执行，除非遇到 break。
  • 使用 continue 可以跳过当前这次循环，直接进入下一轮。

循环还可以返回值：

let result = loop {
    count += 1;
    if count == 10 {
        break count * 2; // 返回 20
    }
};

# 三、while 循环

fn main() {
    let mut n = 3;

    while n > 0 {
        println!("{n}");
        n -= 1;
    }
    println!("起飞！");
}

# 四、for 循环

fn main() {
    let arr = [10, 20, 30];

    for val in arr {
        println!("值: {val}");
    }
}

    println!("{i}");
