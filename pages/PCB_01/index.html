    Layout 01.PCB设计的流程概述

    # 一、Layout 资料准备

    • 原理图：设计文件，用于确认信号流向、电源 / 地网络、器件功能。
    • 结构图：DXF 文件，用于确定板框大小、安装孔位置、器件的高度和排布限制。
    • 设计说明：Layout 设计要求，记录关键信号、器件手册、封装实例、以及推荐布局 / 走线参考。

    # 二、网表导入

    • 顺序：原理图 → PCB
    • 结构文件导入 PCB：默认单位为 mil，单位为 mm 时需注意换算，1 mil = 0.0254 mm。

    # 三、模块规划与按功能规划摆放器件

    • 先确定主控位置或布局，其它模块根据飞线和信号流向进行摆放。

    # 四、器件布局与布局优化

    • 布局顺序：结构器件 → 特殊器件 → 主要器件

    • 原则：先大后小，主要 IC 器件按飞线流向放置合适位置，滤波器件靠近滤波管脚。

    • 优化：器件中心对齐，大器件尽量对齐，相同模块必须一致；注意散热、ESD、静电防护、高压安全距离等。

    # 五、 层叠设置

    • 定义 PCB 层叠结构，每一层对应 叠层名称（如信号层、地层、电源层）。

    # 六、规则设置

    • 线宽 / 线距 / 过孔：根据电流、阻抗、工艺能力设定最小线宽、线距和过孔参数。
    • 整板扇孔：对 BGA 等器件进行整板扇出，确保无走线死角。
    • 电源模块扇孔：对应电源网络，优先处理大电流电源走线。

    # 七、PCB 走线

    • 优先走 重要信号线，重要走线进行过孔地处理，避免电源信号干扰。

    # 八、PCB 修线

    • 布局后进行信号走线，满足阻抗要求，回流路径连续，不存在环形、U 型路径。

    # 九、PCB 等长

    • 高速信号：检查时序、匹配、线长、3W 原则，差分线对差不超过 5mil。

    # 十、布线优化

    • 关键信号：检查关键信号、电源信号载流、路径、过孔数量及处理方式。

    # 十一、DRC 检查

    • 检查是否全部连通，是否存在开路、短路。
    • 检查是否避开禁布区、限高区，确认铣槽、锣铣无误。

    # 十二、丝印处理

    • 丝印方向统一，不跑位、阻焊。
    • 检查丝印是否正确，位号、LOGO 等是否添加。

    # 十三、交付输出

    • ASM：装配图文件。
    • CAM：Gerber 文件，用于制版。
    • SMT：贴片文件表，用于贴片。
    • PRJ：工程文件，源文件。
    • xxx.docx：制版工艺说明文件。
