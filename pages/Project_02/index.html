# 在香橙派上部署每日自动邮件系统（Python + Cron）

作者：liehuf & ChatGPT

在日常学习过程中，保持长期的动力是一件非常重要的事情。与其依赖外界提醒，不如让自己的服务器每天给自己发送一封邮件。这样既可以作为一种激励方式，也可以练习 Linux 自动化工程。

本文记录如何在一台 Orange Pi Zero 3 上部署一个简单的 每日自动发送邮件系统：每天固定时间发送一封包含鼓励内容的邮件。

系统环境：

Orange Pi Zero3
Debian Linux
Python 3.10.12
1
2
3

最终效果是：每天固定时间邮箱会收到一封来自自己服务器的邮件。

# 一、项目原理

整个系统非常简单，由三个部分组成：

Python脚本 —— 负责发送邮件
SMTP服务 —— QQ邮箱提供
cron定时任务 —— Linux定时执行脚本

        ↓

脚本读取鼓励语句
        ↓
通过QQ邮箱SMTP发送邮件
        ↓
邮箱收到每日激励邮件

# 三、安装 pip

部分 Debian / OrangePi 系统镜像默认不带 pip，需要手动安装。

首先更新软件源：

sudo apt-get update
1

然后安装 pip：

sudo apt-get install -y python3-pip
1

安装完成后检查：

pip3 --version
1

如果输出类似：

pip 23.x from /usr/lib/python3/dist-packages/pip
1

说明安装成功。

# 四、更换 pip 国内镜像源

默认情况下 Python 使用官方源，但在国内访问速度非常慢，因此需要更换为国内镜像。

创建配置目录：

mkdir -p ~/.pip
1

创建配置文件：

cat <<EOF > ~/.pip/pip.conf
[global]
timeout = 6000
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn
EOF
1
2
3
4
5
6

这里使用的是 Tsinghua University 提供的 PyPI 镜像源。

验证是否成功：

pip3 config list
1

如果看到：

global.index-url='https://pypi.tuna.tsinghua.edu.cn/simple'
1

说明更换成功。

# 五、创建 Python 虚拟环境

为了避免污染系统 Python 环境，建议使用虚拟环境。

创建虚拟环境：

python3 -m venv venv

source venv/bin/activate

# 八、编写发送邮件脚本

创建脚本：

nano send_mail.py
1

代码如下：

import yagmail
import random
import datetime

sender = "你的QQ邮箱@qq.com"
password = "邮箱授权码"
receiver = "你的QQ邮箱@qq.com"

with open("quotes.txt","r",encoding="utf-8") as f:
    quotes = f.readlines()

quote = random.choice(quotes).strip()

today = datetime.date.today()

subject = f"今日激励 {today}"

content = f"""
今天也要继续前进。

{quote}

—— 来自昨天的自己
"""

yag = yagmail.SMTP(
    user=sender,
    password=password,
    host="smtp.qq.com",
    port=465
)

yag.send(
    to=receiver,
    subject=subject,
    contents=content
)

print("邮件发送成功")

# 十一、设置自动发送（cron）

Linux 使用 cron 实现定时任务。

编辑任务：

crontab -e
1

添加一行：

0 8 * * * /home/orangepi/daily_mail/venv/bin/python /home/orangepi/daily_mail/send_mail.py >> /home/orangepi/daily_mail/log.txt 2>&1
1

含义：

每天 08:00 自动执行脚本
1

时间格式：

分 时 日 月 星期
1

例如：

0 8 * * *    每天8点
0 22 * * *   每天22点
1
2

查看任务：

crontab -l
1

# 十二、常见问题（易错点）

# 1 pip 不存在

部分系统运行：

pip3: command not found
1

解决方法：

sudo apt-get install python3-pip
1

# 2 apt 被锁

有时会出现：

Could not get lock /var/lib/dpkg/lock-frontend
1

原因是系统正在运行：

apt upgrade
1

查看：

ps aux | grep apt
1

一般等待几分钟即可。

# 3 SMTP 登录失败

常见原因：

没有使用授权码
1

必须在 QQ邮箱后台生成 SMTP授权码。

# 4 cron 没执行

可能原因：

虚拟环境路径错误
1

建议使用完整路径：

/home/orangepi/daily_mail/venv/bin/python
1
