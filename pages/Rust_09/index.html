09.Rust的结构体的打印与调试

# Rust的结构体的 println 该如何办？

当我们写了如下代码，

struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let rect = Rectangle {
        width: 20,
        height: 40,
    };

    println!("rect is {}", rect);
}

并且尝试 cargo run时，编译器通常会抱怨：

error[E0277]: `Rectangle` doesn't implement `std::fmt::Display`
1

显然是不能这样玩的，因为编译器并不知道要用什么样的形式把它打印出来。就比如你去汉武大学的图书馆借书，你要是光跟管理员同志说一句借书，显然这位同志只会认为来的是一个糟糕的人。

同理，我们用 Debug 来优雅地打印出结构体。

# 一、使用 {:?} 或 {:#?}

首先，我们在程序开头添加：

#[derive(Debug)]
1

然后，把 println! 宏改成类似于

println!("rect1 is {rect:?}");
1

或者

println!("rect1 is {rect1:#?}");
1

两种结果的输出是有点差异的，读者可以自己调试一哈。

# 二、[dbg! 宏]

没错，还是如下的代码：

#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let scale = 4;
    let rect = Rectangle {
        width: dbg!(20 * scale),
        height: 40,
    };

    dbg!(&rect);
}
