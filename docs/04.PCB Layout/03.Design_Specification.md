## 一、LDO 电源模块

1. 区分输入、输出主回路，滤波电容按**先大后小**原则，紧贴电源芯片输入、输出管脚放置。
2. 为芯片供电的 LDO 需尽量靠近对应芯片，避免低压输出线过长产生压降。
3. 以 LDO 为核心布局，电源滤波器输入、输出端远离布置，防止噪声耦合；元器件整齐紧凑排列，减少、缩短引线与连接。

## 二、USB 接口模块

1. 接口靠近板边放置，便于拔插；ESD 器件紧贴接口，走线需先经过 ESD 器件。
2. 布局时尽量缩短差分线路长度，保证差分对内等长，误差控制在 **±5mil**。
3. 阻抗控制：USB2.0 差分阻抗 90Ω；USB3.0 一般 90Ω，部分场景 100Ω。
4. 差分线打孔包地，换层处加屏蔽地过孔。

## 三、SD/TF 卡模块

1. 电源滤波电容按**先大后小**原则，靠近接口管脚放置；座子靠板边摆放，ESD 器件贴近 TF 卡，走线先经 ESD 器件再接入 SD 卡。
2. 单端阻抗控制为 50Ω，无差分线要求。
3. 所有信号线走同一层，保证信号一致性；与高频信号隔开，空间充足时单根包地，紧张时整组包地，信号线需有完整参考平面。
4. 时钟信号与其他信号线间距保持**15-20mil**，建议全程包地。
5. 信号线总长度控制在 6000mil 内；信号线等长误差 **±100mil**，所有信号与 SD CL 等长误差 **±300mil**。

## 四、Audio 音频模块

### 麦克风部分

1. MIC 靠板边摆放，ESD 器件贴近 MIC，走线先经 ESD 器件再到 MIC 管脚。
2. MIC_N/MIC_P 模拟信号按差分处理，避免同层、临层干扰，远离干扰信号 / 元件；差分对包地，走线宽度建议**10mil 以上**。

### 功放（喇叭）部分

1. 座子靠板边放置，ESD 器件贴近座子；SPEAKER 走线避开敏感信号，不与其他信号 / 电源线隔层 / 同层平行，建议包地，走线宽度**15mil 以上**。
2. 信号输入端用地线屏蔽，且走线尽量短；功放与喇叭间走线**短、直、粗**，抑制电磁辐射。
