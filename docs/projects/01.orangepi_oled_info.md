# 用香橙派 Zero3 + OLED 做一个实时系统状态屏（I2C + Python + 开机自启）

作者：**liehuf & ChatGPT**

---

很多人买单板机都有一个共同的结局：  
**烧一次系统 → 玩两天 → 然后吃灰。**

其实只要做一个**真实世界中一直运行的小项目**，设备基本就不会吃灰了。

本篇文章将手把手完成一个小项目：

> 用 **Orange Pi Zero3 + SSD1306 OLED** 做一个 **系统状态显示屏**

OLED 将实时显示：

```text
OPZero3 In:Y
IP:192.168.x.x
TEMP / CPU （滚动）
TIME / MEM （滚动）
```

效果类似于机构门口的 **LED 信息条**，信息从右向左滚动。

这个小项目会涉及：

- I2C 外设
- Python OLED 驱动
- Linux 系统信息获取
- systemd 开机自启动

最终设备会 **开机自动运行，7×24 小时显示系统状态**。

---

# 一、硬件准备

需要的硬件非常简单：

- Orange Pi Zero3
- SSD1306 OLED（I2C版本）
- 四根杜邦线

常见的 OLED 接口如下：

```text
GND VCC SCL SDA
```

连接方式：

```text
OLED OrangePi Zero3
GND → GND
VCC → 3.3V
SCL → I2C SCL
SDA → I2C SDA
```

---

# 二、确认 I2C 设备

启动 Orange Pi 后，先看看系统里有哪些 I2C 设备：

```bash
ls /dev/i2c*
```

如果看到类似：

```
/dev/i2c-3
/dev/i2c-4
/dev/i2c-5
```

说明 I2C 驱动已经加载。

本项目使用 **i2c-3**。

------

# 三、更新系统

先更新软件源：

```bash
sudo apt update
sudo apt upgrade
```

------

# 四、安装 I2C 工具

安装 I2C 扫描工具：

```bash
sudo apt install i2c-tools
```

扫描设备：

```bash
sudo /usr/sbin/i2cdetect -y 3
```

如果 OLED 正常，会看到：

```bash
30: -- -- -- 3c -- -- -- --
```

`3c` 就是 **SSD1306 的 I2C 地址**。

------

# 五、安装 Python 依赖

安装 pip：

```bash
sudo apt install python3-pip
```

由于 Debian 12 限制 pip 安装，需要加参数：

```bash
pip3 install luma.oled pillow psutil --break-system-packages
```

这些库作用：

| 库        | 作用         |
| --------- | ------------ |
| luma.oled | 驱动 OLED    |
| pillow    | 图像绘制     |
| psutil    | 读取系统信息 |

------

# 六、创建项目目录

```bash
mkdir oled_info_project
cd oled_info_project
```

# 七、编写程序

为了避免 Python 缩进问题，我们直接用 `cat` 写入：

```python
cat << 'EOF' > oled_info.py
from luma.core.interface.serial import i2c
from luma.oled.device import ssd1306
from PIL import Image, ImageDraw, ImageFont
import socket
import time
import subprocess
import psutil

serial = i2c(port=3, address=0x3C)
device = ssd1306(serial)

font = ImageFont.load_default()

def get_ip():
    try:
        s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
        s.connect(("8.8.8.8",80))
        ip = s.getsockname()[0]
        s.close()
        return ip
    except:
        return "No IP"

def internet_ok():
    try:
        socket.create_connection(("8.8.8.8",53),timeout=2)
        return "Y"
    except:
        return "N"

def get_temp():
    temp = subprocess.getoutput("cat /sys/class/thermal/thermal_zone0/temp")
    return str(round(int(temp)/1000,1))+"C"

def get_cpu():
    return str(psutil.cpu_percent())+"%"

def get_mem():
    mem = psutil.virtual_memory()
    used=int(mem.used/1024/1024)
    total=int(mem.total/1024/1024)
    return str(used)+"MB/"+str(total)+"MB"

ip = get_ip()

scroll3 = 0
scroll4 = 0

while True:

    temp = get_temp()
    cpu = get_cpu()
    mem = get_mem()
    now = time.strftime("%H:%M:%S")
    net = internet_ok()

    line3 = " TEMP:"+temp+"   CPU:"+cpu+"   "
    line4 = " TIME:"+now+"   MEM:"+mem+"   "

    image = Image.new("1",(128,64))
    draw = ImageDraw.Draw(image)

    draw.text((0,0),"OPZero3",font=font,fill=255)
    draw.text((80,0),"In:"+net,font=font,fill=255)

    draw.text((0,16),"IP:"+ip,font=font,fill=255)

    draw.text((128-scroll3,32),line3,font=font,fill=255)
    draw.text((128-scroll4,48),line4,font=font,fill=255)

    device.display(image)

    scroll3 += 3
    scroll4 += 2

    if scroll3 > 200:
        scroll3 = 0

    if scroll4 > 400:
        scroll4 = 0

    time.sleep(0.03)

EOF
```

------

# 八、运行程序

```bash
python3 oled_info.py
```

OLED 就会显示：

```text
OPZero3       In:Y
IP:192.168.x.x
TEMP / CPU    ←滚动
TIME / MEM    ←滚动
```

# 九、设置开机自动运行

创建 systemd 服务：

```bash
sudo nano /etc/systemd/system/oled.service
```

写入：

```text
[Unit]
Description=OLED Status Screen
After=network.target

[Service]
ExecStart=/usr/bin/python3 /home/orangepi/oled_info_project/oled_info.py
WorkingDirectory=/home/orangepi/oled_info_project
Restart=always
User=orangepi

[Install]
WantedBy=multi-user.target
```

启用服务：

```bash
sudo systemctl daemon-reload
sudo systemctl enable oled
sudo systemctl start oled
```

重启测试：

```bash
sudo reboot
```

开机后 OLED 会自动运行。

# 十、可能遇到的坑

## 1 Python 缩进错误

常见错误：

```python
IndentationError: unexpected indent
```

原因：Python 对缩进极其敏感。

解决方法：

- 用 `cat << EOF` 写文件
- 或用 VSCode 编辑

------

## 2 I2C 权限问题

错误：

```bash
Permission denied: /dev/i2c-3
```

解决：

```bash
sudo usermod -aG i2c orangepi
```

重新登录即可。

------

## 3 找不到 i2cdetect

可能出现：

```bash
i2cdetect: command not found
```

原因是路径问题。

使用完整路径：

```bash
sudo /usr/sbin/i2cdetect -y 3
```

------

## 4 pip 安装失败

Debian12 会报：

```bash
externally-managed-environment
```

解决：

```bash
pip3 install xxx --break-system-packages
```
